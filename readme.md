# Dasar Analisis Data Menggunakan Python

## Setup environment
```
python3 -m virtualenv .env
source .env/bin/activate
pip install requirements.txt
```

## Run steamlit app
```
streamlit run dashboard/dashboard.py
```

## Deployed URL
```
https://anhilmy-python-analysis.streamlit.app/
```
